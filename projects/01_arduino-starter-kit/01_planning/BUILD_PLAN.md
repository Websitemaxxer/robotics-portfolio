# Build Plan — Arduino Starter Kit

A rough plan, grouped by theme rather than strict weeks. It **will** change — that is expected and good.

| Stage | Goal | Done? | Notes |
|------|------|:-----:|-------|
| 1 | Basics of output & input — LEDs and the pushbutton (Projects 1–2) | ◑ | Project 2 (Spaceship Interface) done 2026-07-06; took 3 wiring faults to get right |
| 2 | First sensors — temperature & light (Projects 3–6) | ☐ | Introduces `analogRead`, the TMP36, and an LDR |
| 3 | Sound, servos & motors (Projects 5–10) | ☐ | Servo, DC motor + transistor, piezo |
| 4 | Bigger builds — LCD, keyboard, knock lock (Projects 11–15) | ☐ | More components, more logic |
| 5 | My own extensions (≥ 3 across the projects) | ☐ | Labelled `*_mine.ino`; first target = variable-speed Spaceship alarm |

> **The gap between this plan and what actually happened is itself good evidence.**
> When reality differs from the plan, don't erase the plan — note what changed in the
> "Notes" column and explain *why* in the [build diary](BUILD_DIARY.md). Universities
> and engineers want to see how you adapt, not a plan that magically went perfectly.
